# First login troubleshooting

If first login fails, walk the customer through these checks in order before
escalating. 90% of first-login issues are one of these.

## Checks
1. Confirm they are using the invite link, not the general login page.
2. MFA: the invite email prompts them to enroll in MFA on the first login.
   If they skipped it, the second login will force it.
3. Browser cookies: Safari in private mode blocks our session cookie. Ask
   them to try Chrome or a non-private window.
4. Confirm the invite has not expired. Invites are valid for 7 days.
